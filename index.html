Monitoring space weather to protect vulnerable people

Мониторинг космической погоды для защиты метеозависимых

SolarWind Dashboard collects and visualizes NASA solar wind and geomagnetic data in near real-time, issues alerts, and provides interactive analytics intended to inform medical-vulnerable populations, operators and researchers.

SolarWind Dashboard собирает и визуализирует данные солнечного ветра и геомагнитные индексы (NASA) в режимe почти реального времени, отправляет алерты и предоставляет инструмент для анализа — полезный для метеозависимых людей и исследователей.

About the project

О проекте

SolarWind Dashboard is designed to translate raw space weather telemetry into human-actionable information. The system focuses on:

SolarWind Dashboard предназначен для преобразования сырых данных космической погоды в информацию, пригодную для использования людьми. Система фокусируется на:

  • protecting health-sensitive populations (cardiac patients, elderly)
  • providing transparent reproducible data processing for researchers
  • creating a polished academic case suitable for graduate admissions
  • защите метеозависимых людей (сердечные больные, пожилые)
  • предоставлении прозрачной воспроизводимой обработки данных для исследователей
  • создании качественного академического кейса для поступления в аспирантуру

Data & reproducibility

Данные и воспроизводимость

Data sources:

Источники данных:

  • NASA/OMNI, ACE and DSCOVR solar wind products
  • Kp / Dst indices (space weather centers)
  • Historical archives for case studies
  • Продукты NASA/OMNI, ACE и DSCOVR по солнечному ветру
  • Индексы Kp / Dst (центры космической погоды)
