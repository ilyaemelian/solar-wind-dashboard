About the project

SolarWind Dashboard is designed to translate raw space weather telemetry into human-actionable information. The system focuses on:

  • protecting health-sensitive populations (cardiac patients, elderly)
  • providing transparent reproducible data processing for researchers
  • creating a polished academic case suitable for graduate admissions
